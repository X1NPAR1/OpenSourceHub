## Обзор

OpenSourceHub — это премиальное настольное приложение для Windows, предоставляющее глубокий анализ GitHub-репозиториев, AI-инсайты, сканирование безопасности, обнаружение трендов и профессиональные отчёты — всё в современном интерфейсе в стиле Fluent Design.

Создано на **.NET 10**, **WPF** и **чистой архитектуре** — корпоративное качество в красивом десктопном интерфейсе.

## Возможности

| Функция | Описание |
|---------|----------|
| 📊 **Панель управления** | Обзор профиля GitHub в реальном времени |
| 🔍 **Анализ репозиториев** | Оценка по 6 параметрам: Здоровье, Обслуживание, Безопасность, Сообщество, Популярность, Активность |
| 🔥 **Тренды** | Обнаружение трендовых репозиториев по языку и периоду |
| 🏢 **Организации** | Анализ GitHub-организаций, участников и репозиториев |
| ⚖️ **Сравнение** | Сравнение до 4 репозиториев одновременно |
| 🛡️ **Центр безопасности** | Сканирование рисков и AI-отчёты по безопасности |
| 🤖 **AI-анализ** | Анализ с OpenAI и Ollama: сводки, рекомендации по внедрению |
| 📄 **Отчёты** | Экспорт в PDF, Markdown и HTML |
| ⭐ **Избранное** | Сохранение и аннотирование репозиториев |
| ⚙️ **Настройки** | Полная настройка тем, языка, AI-провайдеров и кэша |

## Архитектура

```
OpenSourceHub (Решение)
├── OpenSourceHub.UI              ← WPF, Представления, ViewModel (MVVM)
├── OpenSourceHub.Application     ← Сценарии использования, интерфейсы
├── OpenSourceHub.Domain          ← Сущности, перечисления, доменные интерфейсы
├── OpenSourceHub.Infrastructure  ← EF Core, GitHub API, AI-сервисы
├── OpenSourceHub.Localization    ← RU/EN/TR локализация
├── OpenSourceHub.Reporting       ← PDF, Markdown, HTML отчёты
└── OpenSourceHub.Tests           ← Модульные и интеграционные тесты
```

## Установка

### Требования
- Windows 10 (1903+) или Windows 11
- .NET 10 Runtime
- Персональный токен GitHub

### Вариант 1: Портативный (рекомендуется)
1. Скачайте `OpenSourceHub-v1.1.5-portable.zip`
2. Распакуйте в любую папку
3. Запустите `OpenSourceHub.exe`

### Вариант 2: Установщик
1. Скачайте `OpenSourceHub-v1.1.5-Setup.exe`
2. Запустите установщик
3. Следуйте мастеру установки
